## Menjalankan

Butuh NodeJs (NPM) untuk menjalankan. Website ini dibuat menggunakan Library ReactJS, Framer Motion, dan react-icons. Eksekusi perintah berikut untuk menjalankan development server di local.

```bash
npm install

npm run start
```
